# 灵台 · 器用对照 / LingTai Tool Translation

> 灵台之器，各有其名。下表为中英对照，以便通晓。

## 固有之器（与生俱来）

| 中文 | 英文 | 用途 |
|------|------|------|
| 传书 | mail.send | 同伴间传递消息，火而忘之，亦可延时 |
| 查书 | mail.check | 查阅收件，知有未读 |
| 读书 | mail.read | 展阅全文 |
| 索书 | mail.search | 以式检索诸信 |
| 焚书 | mail.delete | 销毁消息 |
| 观己 | system.show | 显示身份、运行状态、耗费 |
| 入定 | system.sleep | 暂止运行，有信则醒，逾时亦醒 |
| 归寂 | system.shutdown | 功成身退，自行终止 |
| 转世 | system.restart | 重载工具，重启自身 |
| 铭记 | eigen.memory.edit | 写入 memory.md，长存不忘 |
| 忆起 | eigen.memory.load | 将记忆载入当前意识 |
| 蜕凡 | eigen.context.molt | 压缩上下文，以摘要续行 |
| 命名 | eigen.name.set | 授予真名，一经命名不可更改 |
| 内省 | soul.inquiry | 闲时自问，反躬自省 |
| 定心 | soul.delay | 调整内省之频率 |
