# 灵台 · 工具对照表

> 以下为中英文工具名对照，方便使用。

## 固有工具（所有代理自带）

| 中文 | 英文 | 用途 |
|------|------|------|
| 发信 | mail.send | 向同伴发送消息，可延时发送 |
| 查信 | mail.check | 查看收件箱，显示未读 |
| 读信 | mail.read | 阅读消息全文 |
| 搜信 | mail.search | 按正则表达式搜索消息 |
| 删信 | mail.delete | 删除消息 |
| 查看状态 | system.show | 显示身份、运行状态、资源消耗 |
| 休眠 | system.sleep | 暂停运行，收到消息或超时后唤醒 |
| 关闭 | system.shutdown | 自行终止运行 |
| 重启 | system.restart | 重新加载工具并重启 |
| 编辑记忆 | eigen.memory.edit | 写入 memory.md，持久保存 |
| 加载记忆 | eigen.memory.load | 将记忆载入当前上下文 |
| 压缩上下文 | eigen.context.molt | 压缩上下文，以摘要继续 |
| 命名 | eigen.name.set | 设定真名，设定后不可更改 |
| 自省 | soul.inquiry | 空闲时自我审视 |
| 调节自省频率 | soul.delay | 调整自省的时间间隔 |

## 可选工具（按需启用）

### 文件操作

| 中文 | 英文 | 用途 |
|------|------|------|
| 读取 | read | 读取文件内容 |
| 写入 | write | 创建或覆写文件 |
| 编辑 | edit | 精确替换文件中的内容 |
| 查找文件 | glob | 按模式查找文件 |
| 搜索内容 | grep | 按正则表达式搜索文件内容 |
| 执行命令 | bash | 执行 shell 命令 |

### 心印与记忆

| 中文 | 英文 | 用途 |
|------|------|------|
| 更新心印 | psyche.character.update | 修订自己的心印（character.md） |
| 加载心印 | psyche.character.load | 将心印载入当前上下文 |
| 构建记忆 | psyche.memory.construct | 用知识条目和文本构建记忆 |
| 加载记忆 | psyche.memory.load | 将记忆载入当前上下文 |
| 压缩上下文 | psyche.context.molt | 压缩上下文，以摘要继续 |
